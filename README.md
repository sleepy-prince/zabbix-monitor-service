# zabbix-monitor-service

Данный проект представляет собой локальный отказоустойчивый стек мониторинга на базе **Zabbix 7.0**. Стек развернут в контейнерах Docker и настроен на сбор метрик хост-машины в безопасном активном режиме.

## Архитектура решения

1. **Zabbix Database (PostgreSQL 17):** Хранилище данных, изолированное во внутренней сети Docker. Снабжено автоматической проверкой здоровья (`healthcheck`).
2. **Zabbix Server:** Главный вычислительный движок. Ожидает готовности базы данных перед запуском.
3. **Zabbix Web UI (Nginx):** Панель управления.
4. **Zabbix Agent 2:** Установлен на хостовой системе. Собирает метрики железа и отправляет их на сервер в активном режиме.
